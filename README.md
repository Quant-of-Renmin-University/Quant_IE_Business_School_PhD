## Course Description
This course is designed to train PhD students to use the artificial intelligence and Python programming for the financial predictions. The course will cover the knowledge of machine learning, deep learning, and natural language processing. Students will learn how to use Python to write code and how to use Python libraries to build machine learning models. 

We will meet at every week:
- Velazquez, 130 V407 from 11am to 2pm in the 2026 Spring Semester 

## Contents:
1. Python Basics 
2. Financial Big data (Kaggle, HuggingFace, WRDS, CSMAR)
3. Machine Learning Fundamentals (Optimization, Lasso, Emsembled Methods, Modeling Emsembling)
4. Deep Learning Fundamentals (ANN, CNN)
5. Natural Language Processing Fundamentals (Transformers)
6. Papers Reading (XGBOOST Chen paper, Transformer Vaswai paper, Two top finance papers using ML)
